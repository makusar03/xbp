Git

Git:誰がいつどの様に編集したかを正確に把握できる分散型バージョン管理システム
＜特徴＞
  • 古いバージョンに簡単に戻せる
  • 新旧のファイルを一元管理できる
  • 編集した履歴を複数人で共有できる
  • 複数人で修正した部分を一つに統合できる

HTML

HTMLとは、インターネットで閲覧しているwebサイトに表示される画面をつくっている言語である！

超簡単にいうと、ホームページファイルの書き方のルール！
Hyper　Text Markup Languageを略したものでめっちゃすごいテキストである！
htmlとは人間側に取って見やすい情報表示を行うためのものだが、xmlは機械側にとって分かりやすく情報伝達するためのものである。

HTMLタグの代表的な種類

＜h１＞〜＜h６＞ 見出しを作成するタグ（数字が小さいほど大きい見出し）
＜a＞ リンクを埋め込むタグ
＜p＞ 段落指定するタグ
＜br＞ 改行指定するタグ
＜img＞ 画像を挿入するタグ
＜button＞ ボタンを作成するタグ
